# SolaEngine
写了一个简单的2D引擎。</br>

</br>
引擎目前包含功能：</br>
	场景树渲染与处理</br>
	半透明渲染处理</br>
	世界坐标与模型坐标相互转换</br>
	屏幕点击事件</br>
	节点的Update事件</br>
	事件Event中心</br>
</br>
后续需要加上的功能：</br>
	文字渲染</br>
	点击冒泡处理</br>
	音效模块</br>
</br>
由于历史原因，实现部分得比较粗暴，后面会慢慢改进和优化代码。</br>
